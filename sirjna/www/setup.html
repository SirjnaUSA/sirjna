Sirjna — Setup Guide

One-time steps

  • Desk → Email Account → add contact@sirjna.com
  • Website Settings → set theme; Portal Settings → Default Home = /portal (optional)
  • Create DocTypes via Desk (no console needed)
  • Publish public pages (Home, Pricing, Webinars) as needed

This MVP contains only the essentials so it installs cleanly on Frappe Cloud.
